# 创建你的第一个创新项目

本页面将一步一步指导你，如何在创新 GitHub 组织中创建一个**新的创新项目 Repo**。

不需要 GitHub 经验，不需要提前准备完整方案。

---

## ✅ 在开始之前，你需要具备什么？

请确认你已经满足以下条件：

- ✅ 已加入创新 GitHub Organization
- ✅ 已加入 `incubator-creators`（或类似）团队
- ✅ 你有一个想尝试的想法（哪怕只有一句话）

> 💡 不需要是完整方案  
> “我想试试用 AI 自动生成会议总结” 就足够了

---

## 🚀 Step 1：使用创新项目模板创建 Repo

1. 打开创新 GitHub Organization 首页
2. 点击 **New repository**
3. 选择模板仓库：
   - `template-innovation-repo`
4. 填写仓库信息：
   - Repository name：项目名称（英文或拼音）
   - Description：一句话说明你要做什么
5. 点击 **Create repository**

✅ 到这里，你已经有了一个属于你的创新项目 Repo。

---

## 📝 Step 2：完善项目 README（非常重要）

进入你刚创建的项目 Repo，打开 `README.md`，你会看到模板内容。

请重点填写以下几部分：

### 1️⃣ Problem（问题）
- 你想解决什么问题？
- 谁会遇到这个问题？

### 2️⃣ Idea（想法）
- 你的初步解决思路是什么？
- 为什么你觉得它值得尝试？

### 3️⃣ Status（状态）
- 勾选当前阶段：
  - idea / prototype / active

---

## 👥 Step 3：邀请协作者（可选）

如果你不是一个人做项目：

1. 进入 Repo → **Settings**
2. 点击 **Collaborators**
3. 添加同事 GitHub 账号

> 💡 创新项目鼓励小团队协作，而不是单打独斗

---

## 🧪 Step 4：开始尝试（不要等完美）

现在你可以：

- 写第一版代码
- 画一个流程图
- 用 Copilot / Agent 做一个 Demo
- 记录失败和学习

✅ **完成 > 完美**

---

## 📈 下一步可以做什么？

当你的项目逐渐清晰后，你可以：

- 更新 README 的 Status
- 在 Demo Day / Hackathon 中展示
- 邀请更多人试用和反馈

如果项目成熟，可以进入 **Graduation（毕业）流程**。

👉 参见：LIFECYCLE.html

---

## ✅ 你已经完成了什么？

如果你走到这里，说明你已经：

- ✅ 创建了一个创新项目 Repo
- ✅ 明确了问题和想法
- ✅ 正式开始了一次创新尝试

🎉 **欢迎来到创新旅程**
