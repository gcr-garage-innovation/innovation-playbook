💡 Idea — 用一句话写下你想解决的问题，创建 Repo 并标记 idea。

🧪 Prototype — 快速构建 PoC / Demo，用最小成本验证可行性。

🚀 Active — 有稳定使用场景和小范围用户，持续收集反馈。

🎓 Graduation — 项目成熟、业务价值明确，迁入正式产品线。详细了解 →

💡 Idea — Write down the problem you want to solve in one sentence, create a Repo and tag it idea.

🧪 Prototype — Quickly build a PoC / Demo to validate feasibility at minimal cost.

🚀 Active — Stable use cases with a small user base, continuously collecting feedback.

🎓 Graduation — Project is mature with clear business value, migrated to production.Learn More →
