# 快速开始创新项目

> 🎯 适合谁：第一次接触创新项目的员工
> 📌 本页面解决什么：如何从 0 开始参与创新  
> 👉 阅读后你能做什么：知道第一步该做什么，不再迷茫

## 第一步：加入创新 GitHub 组织
- 联系管理员加入组织
- 加入 incubator-creators Team

## 第二步：创建创新 Repo
- 使用 template-innovation-repo
- 填写 README 中的问题定义

## 第三步：开始尝试
- 不需要完美
- 快速 PoC / Agent / Demo
